crypto-algorithms
=================


About
---
These are my implementations of some of Brad Conte's cryptography algorithms, adapted to work with [hash-distrib](https://github.com/rimopa/hash-distrib).

This code is released into the public domain free of any restrictions. The author requests acknowledgement if the code is used, but does not require it. This code is provided free of any liability and without any quality claims by the author.

Note that these are *not* cryptographically secure implementations. They have no resistence to side-channel attacks and should not be used in contexts that need cryptographically secure implementations.

Build and test information can be found on [hash-distrib](https://github.com/rimopa/hash-distrib).
